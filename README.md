# Dotfiles README


## Requied

* MacOS
* install brew
	*  see: https://brew.sh/index_ja.html

## install

### install depended packages
```
$ brew install reattach-to-user-namespace
$ brew install tmux
```

### get dotfiles

```
$ cd ~
$ git clone https://kamorita@bitbucket.org/kamorita/.dotfiles.git
$ ln -s .dotfiles/.tmux.conf .tmux.conf
$ ...
```
